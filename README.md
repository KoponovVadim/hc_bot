# Hangcha B2B Chat-Funnel Widget

Интерактивный чат-виджет для подбора вилочных погрузчиков и складской техники Hangcha для B2B-сайта.

## Особенности
- Чистый HTML + CSS + JS, без фреймворков
- Строгий корпоративный стиль
- Многоуровневая воронка с ветвлением сценариев
- Валидация контактов (телефон, email)
- Итоговая сводка заявки
- Полная поддержка мобильных устройств

## Как использовать
1. Скопируйте файл `bot.html` на ваш сайт или сервер.
2. Откройте файл в браузере — чат-виджет появится в правом нижнем углу.
3. Для интеграции на сайт вставьте содержимое `<body>` и `<script>` в нужное место страницы.

## Сценарии чата
- Подбор погрузчика (с выбором типа двигателя, условий, параметров)
- Проверка наличия техники
- Подбор складской техники
- Запрос сервисных услуг, АКБ, запчастей
- Свободный вопрос менеджеру

## Структура
- `bot.html` — основной файл с разметкой, стилями и логикой чата
- `README.md` — данная инструкция
